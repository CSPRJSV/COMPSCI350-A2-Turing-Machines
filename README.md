【重点解析:】

Q1: 设计一个能执行 Reverse Ternary Addition 的图灵机，要画图和描述。Reverse Ternary 就是倒三进制表示法，比如 Reverse Ternary 的数 102 在正常三进制表示下是 201，也就是 2*3^2+0*3^1+1*3^0 = 19（十进制）。题目要求使用 Reverse Ternary 实际上降低了难度

Q2: 证明一个变种图灵机（纸带左右皆可无限延伸）与标准图灵机的等价性。Idea 不难想，lecture 里其实有提到，难度在于描述详细具体的构造方式

Q3: 考察对 Turing-decidable 和 Turing-recognisable languages 的 closure property 的灵活运用，不是让直接证明。

Q4: 作业给出了 ChatGPT 对于 “如果图灵机纸带字符集可包含全体实数，那么将会怎样改变它们的计算能力？” 这一问题的两段回答，要求阅读并给出你的批判。这题比较反常规，目的是考察对 Turing Machines 概念的深层次理解，防止死记硬背，因为只有真正理解 Turing Machines 的工作机制才能合理推测其某部分发生改变所带来的后果。
